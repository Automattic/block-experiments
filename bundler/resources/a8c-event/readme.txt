Feature an event in your post

== Description ==

Let everyone know about the events going on. Whether you're hosting or attending, the event block will feature your event including the time and location.

## Source and Support

You can follow development, file an issue, suggest features, and view the source at the Github repo: <a href="https://github.com/Automattic/block-experiments">https://github.com/Automattic/block-experiments</a>
